#🌱 Sustentabilidade & Consumo Sustentável
Sustentabilidade: É o equilíbrio entre o desenvolvimento econômico, o bem-estar social e a preservação ambiental, garantindo que os recursos do planeta durem para as próximas gerações.

Consumo Sustentável: É o ato de consumir com consciência. Significa comprar o necessário, escolher produtos com menor impacto ecológico, reduzir o desperdício e adotar a prática dos 5 Rs (Recusar, Reduzir, Reutilizar, Reciclar e Responsabilizar-se).
